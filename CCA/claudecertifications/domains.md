# Claude Certified Architect Exam Domains

## Overview

The Claude Certified Architect – Foundations exam evaluates knowledge across five core domains. Each domain focuses on specific competencies needed for production-grade AI application development.

## The 5 Exam Domains

### Domain 1: Agentic Architecture & Orchestration (~25%)

This domain covers designing and implementing intelligent agent systems using Claude's Agent SDK. Key topics include:

- Agentic Loops & Core API
- Multi-Agent Orchestration
- Hooks & Programmatic Enforcement
- Session Management & Workflows

Candidates learn to build "production-grade AI applications" using established patterns for task decomposition and multi-agent coordination.

### Domain 2: Tool Design & MCP Integration (~20%)

Focus areas for effective tool implementation and Model Context Protocol integration:

- Tool Description Best Practices
- Structured Error Responses
- Tool Distribution & Selection
- MCP Server Configuration
- Built-in Tools

This domain emphasizes designing tools that Claude can reliably interact with in complex workflows.

### Domain 3: Claude Code Configuration & Workflows (~20%)

Development workflow optimization through:

- CLAUDE.md Hierarchy & Configuration
- Custom Commands & Skills
- Plan Mode & Iterative Refinement
- CI/CD Integration & Batch Processing

The domain explores configuring Claude Code for seamless developer integration.

### Domain 4: Prompt Engineering & Structured Output (~20%)

Production-ready prompting techniques including:

- Explicit Criteria & Instruction Design
- Few-Shot Prompting
- Tool Use for Structured Output
- Validation-Retry Loops & Multi-Pass Review

Topics address "JSON schema design" and systematic output validation.

### Domain 5: Context Management & Reliability (~15%)

Managing context in production systems covers:

- Context Optimization & Positioning
- Escalation & Error Propagation
- Context Degradation & Extended Sessions
- Human Review & Information Provenance

## Study Resources

A structured 12-week study plan is available covering all domains in optimal sequence with daily tasks and practice assessments.
