# Claude Certifications - Free Courses & Exam Prep

## Overview

Claude Certifications offers free learning resources focused on AI development, prompt engineering, and professional certification preparation for the Claude AI platform.

## Main Course Offerings

**Claude 101: Complete Beginner's Guide**
- 13 lessons, 3 hours
- Covers fundamentals: prompting, projects, artifacts, skills, connectors, and research mode
- Beginner level

**Claude Code in Action**
- 15 lessons, 6 hours
- Topics include context management, hooks, MCP servers, GitHub integration, and SDK
- Intermediate level

**AI Fluency Foundations**
- 13 lessons, 4 hours
- Teaches the "4D Framework" (Delegation, Description, Discernment, Diligence)
- Focus on effective human-AI collaboration
- Beginner level

**Claude Certified Architect – Foundations**
- Exam preparation materials
- 25+ practice questions with explanations
- 12-week study plan
- Advanced level
