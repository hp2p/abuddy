# Claude Certified Architect – Foundations (CCA-F): 2026 Exam Guide

## Overview

Anthropic launched the Claude Certified Architect – Foundations (CCA-F) exam on March 12, 2026. This is the company's first official technical certification for production-grade Claude application development.

**Exam Specifications:**
- Format: 60 scenario-based multiple-choice questions
- Duration: 120 minutes (proctored, no breaks)
- Passing Score: 720/1000
- Delivery: ProctorFree platform
- Results: 2 business days with detailed breakdown

## Five Weighted Domains

| Domain | Weight | Focus |
|--------|--------|-------|
| Agentic Architecture & Orchestration | 27% | Multi-agent systems, task decomposition |
| Claude Code Configuration & Workflows | 20% | CLAUDE.md files, custom commands, CI/CD |
| Prompt Engineering & Structured Output | 20% | Advanced prompting, JSON mode, tool calling |
| Tool Design & MCP Integration | 18% | Model Context Protocol servers |
| Context Management & Reliability | 15% | Long-context strategies, reliability patterns |

## Eligibility & Cost

**Current Access:** Exclusive to Anthropic Partner Network members (free to join)

- First 5,000 partner employees: **Free exam**

Requires completion of Anthropic Academy 200-level courses covering Claude API, Agent SDK, and MCP fundamentals.

## Preparation Strategy

1. **Join Claude Partner Network** (free at claude.com/partners)
2. **Complete official Academy courses:**
   - Claude 101
   - Building with the Claude API
   - Claude Code in Action
   - Introduction to Model Context Protocol

3. **Study the Exam Guide** (provided in Skilljar)
4. **Take practice exam** (target: 900+/1000)
5. **Focus heavily on agentic architecture** (27% weight, highest difficulty)

**Typical prep time:** 2-4 weeks, 20-30 hours for experienced Claude developers

## Why It Matters

This credential demonstrates production-level architectural capability—distinct from general AI literacy certifications. Major consulting firms like Accenture are already training thousands of professionals, positioning CCA-F as the AWS Solutions Architect equivalent for the Claude ecosystem.

Early adoption offers competitive advantage before widespread credential saturation.
